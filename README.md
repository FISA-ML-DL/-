# 💳 채무 불이행 여부 예측 해커톤: 불이행의 징후를 찾아라!
이 프로젝트는 사용자 데이터를 기반으로 소비 패턴을 분석하고, 이를 바탕으로 맞춤형 추천을 제공하는 시스템입니다. 
<br>소비 데이터는 카드 사용 내역을 포함하고 있으며, 사용자의 연령대, 성별, 라이프스타일에 맞춘 분석을 통해 유용한 인사이트를 제공합니다.
